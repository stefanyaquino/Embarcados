2. Defina qual sistema operacional você utilizará no projeto desta disciplina com o Raspberry Pi. Justifique sua escolha.
Raspbian, porque é um sistema "padrão" para quem está tendo o primeiro contato com a placa, proporcionando uma maneira mais fácul de utilizar.

3. Defina de qual forma você instalará o sistema operacional escolhido. Escreva o passo-a-passo da instalação e forneça os links necessários para isto.
Via SD card. Primeiramente, deve-se baixar o sistema operacional no forma .zip no site <https://www.raspberrypi.org/downloads/noobs/>. Então, após o download, esse arquivo deve ser descompactado. Todos os arquivos presentes devem ser copiados para um cartão de memória de no mínimo 4Gb. Feito isso, o cartão deve ser inserido na raspberry, e ela deve ser ligada à fonte e a um monitor, para proceder com a instalação. Então, deve-se selecionar o Raspnian e instalar.
